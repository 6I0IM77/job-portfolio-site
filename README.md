# 刘立伟个人工作求职网站

这是一个可直接部署到 GitHub Pages 的静态个人求职网站，包含：

- `index.html`：网站主页
- `assets/css/style.css`：页面样式
- `assets/js/main.js`：导航与联系表单交互
- `assets/img/hero-workspace.png`：首屏视觉图

## 部署到 GitHub Pages

1. 新建一个 GitHub 仓库。
2. 把本目录里的全部文件上传到仓库根目录。
3. 进入仓库 `Settings` → `Pages`。
4. `Source` 选择 `Deploy from a branch`。
5. `Branch` 选择 `main`，目录选择 `/root`，保存。
6. 等待 GitHub 生成访问地址。

## 需要替换的个人信息

- `index.html` 里的邮箱：`your-email@example.com`
- `index.html` 里的 GitHub：`github.com/your-github`
- 联系电话、目标岗位、项目经历可以按真实情况继续修改。
